# casa_B07-09_sales_website

## 🏢 販売用ランディングページ

このプロジェクトは、Vite + React + TypeScriptで構築された販売用ランディングページです。

### 🌐 公開URL

**GitHub Pages**: https://asecro.github.io/casa_B07-09_sales_website/

### 🚀 セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

# GitHub Pagesへのデプロイ
npm run deploy
```

### 📁 プロジェクト構造

```
/
├── src/           # ソースコード
├── build/         # ビルド出力
├── index.html     # エントリーポイント
├── package.json   # 依存関係
└── vite.config.ts # Vite設定
```

### 🛠 技術スタック

- **Vite** - ビルドツール
- **React** - UIフレームワーク
- **TypeScript** - 型安全な開発
- **Tailwind CSS** - スタイリング
- **Radix UI** - UIコンポーネント
